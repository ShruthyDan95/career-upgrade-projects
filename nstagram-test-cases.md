Test Cases – Instagram Login Page

1. Test Case: Login with valid username and valid password

Steps:
1. Open Instagram login page
2. Enter a valid username
3. Enter a valid password
4. Click “Log in”

Expected Result: User should successfully log in and reach the home page.

2. Test Case: Login with valid username and invalid password
Expected Result: Error message should appear: “Incorrect password.”

3. Test Case: Login with empty username and password fields
Expected Result: “Username required” or “Password required” message should appear.

4. Test Case: Login with invalid username format (e.g., missing '@')
Expected Result: Validation message should appear or login should fail.

5. Test Case: Check “Forgot Password” link
Expected Result: User should be redirected to password reset page.
